Digital Literacy Project Report

1. Introduction

Before I started this project I thought digital literacy meant knowing how to use a phone or basic apps.. While working on it I realized it's much more. It's about using technology presenting ourselves online and handling things safely.

This project let me explore platforms I'd heard of but never used properly. I created accounts, uploaded work. Understood how they work in real life. Some parts were confusing at first. Everything made sense eventually. By the end I felt more confident using these platforms. Understood why they're important for students.

2. Objectives

The main goal of this project wasn't just to complete it for marks but to learn something for the future. Some objectives were:

* To understand what digital literacy means in life

* To learn how to use platforms like GitHub, LinkedIn and Kaggle

* To build a digital identity

* To understand how students can present themselves online

* To learn online safety practices

3. Platforms and Tools Used

### GitHub

At first GitHub seemed complicated. There were terms like repository, commit, README, which I didn't understand.. After trying it I created a repository and uploaded my project. Writing the README file was a learning experience because I had to explain my work

Now I think GitHub is very useful for students working on technical projects.

### LinkedIn

Creating my LinkedIn profile was different from what I do online. I had to think what to write what skills to add, how to present myself. It made me realize how we show ourselves matters. This platform can be helpful in the future for internships and opportunities.

### Kaggle

Kaggle was new to me. I didn't understand much at first. I explored different datasets. Even though I'm still learning it gave me an understanding of how data is used. It also made me interested in learning more about data science.

4. Work Done in This Project

In this project I focused on building my presence step by step. First I created my GitHub account. Explored how it works. I created a repository. Uploaded my project files. Writing the README file took time because I wanted it to be clear.

Then I created my LinkedIn profile. I added my education details, skills and interests. I tried to make it look proper and meaningful. After that I created my Kaggle account. Explored datasets. I understood the basics. How people use data in projects.

Along with this I learned about safety. Before this I didn't think much about passwords or suspicious links. Now I understand how important they are.

Some things I became more careful about:

* Not using passwords

* Not clicking random links

* Being careful about what I share

5. Key Learnings

This project taught me more than I expected. I learned how GitHub works. I understood how to write a README file. LinkedIn helped me present myself professionally. Kaggle gave me an introduction to the world of data.

One of the most important things I learned was how to be responsible online. Small habits can make a difference.

6. Challenges Faced

There were challenges. In the beginning GitHub was confusing. I didn't understand how to upload files. The README file also took time. LinkedIn was new. I had to think more about presenting myself. Kaggle felt advanced. I still explored it.

With time and practice these things became easier.

7. Conclusion

Looking back this project was useful. It wasn't about completing an assignment but about learning something practical for the future. Now I feel more comfortable using platforms like GitHub and LinkedIn.

I also feel more aware about how to behave and manage my digital presence.

This project gave me a starting point. I think it will help me as I learn more in my field.

8. Future Scope

There's still a lot I want to improve. I plan to upload projects on GitHub and make my profile stronger. I also want to improve my LinkedIn profile by adding achievements and certifications.

On Kaggle I want to explore more and maybe try working on datasets. I'm also interested, in learning more about AI and Machine Learning so this project feels like a step.
